# Texture 纹理

## 1. 概念

- 纹理：  
   纹理是贴在 Mesh 表面的那一层皮，可以是静态图片 image（位图），也可以是由算法生成的 Patten （图案、花样）

- UV 贴图：  
   “UV”是纹理贴图坐标的简称。 它定义了图片上每个点的位置的信息。 这些点与 3D 模型是相互联系的，以决定表面纹理贴图的位置。 就好像平面的布料，剪裁制作成立体的衣服一样。  
   UV 贴图就是将纹理二维坐标映射（Map）到 3D 网格上的过程。
