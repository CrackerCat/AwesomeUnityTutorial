# 管理屏幕尺寸和锚点

## 1. 管理屏幕尺寸

### 1.1 屏幕横纵比

横纵比（宽高比）描述了屏幕的宽度和高度之间的关系

例如，老式电视和最早的电影使用 4:3 的纵横比。这意味着屏幕有 4 个单位宽和 3 个单位高。然而，现代宽屏电视和大多数新电影使用更宽的 16:9 纵横比，而手机的默认通常为 9：16 或 10：16
